Getting Started

Follow these steps to run Cipher on your device:


  1. Clone the Repository
    Use Git to clone the Cipher repository: 
    git clone https://github.com/pappricot/cipher.git
    Then navigate to the project directory: 
    cd cipher
  2. Install Dependencies
    Install the required Python packages: 
    pip install -r requirements.txt
  3. Download Language Models
    Run the setup script to download necessary models (requires internet for first-time setup): 
    python setup_models.py
    Models are stored locally after download and used offline thereafter.
  4. Run Cipher
    Start the application: 
    python main.py

Notes: Ensure you have Python 3.11+ and Git installed. For advanced users, optional Docker instructions are available in the GitHub README.
